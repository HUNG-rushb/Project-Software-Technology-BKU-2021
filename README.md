## General info
Dự án ứng dụng bán hàng e-commerce Nhà hàng Món Việt.
	
## Technologies
Dự án dược tạo thành với những công nghệ sau:
* VueJS 3.0
* Bootstrap 5
* Firebase Firestore
* Zalo chat
* MoMo QR Pay
	
## Setup
Để chạy thử dự án, sau khi download code về:

```
$ cd ./project
$ npm install
$ npm run serve
```

Dự án sẽ chạy trên localhost:8080/
